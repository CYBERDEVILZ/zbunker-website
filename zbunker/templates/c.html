Introduction to c

c ("Network Mapper") is a free and open source (license) utility for network discovery and security auditing. Ethical Hackers and Bug Bounty Hunters use this tool to gather as much info about the network as possible. Its literally the most widely sought after tool for reconnaissance in a network.
